= Edit
:toc: right
:imagesdir: images

The *Edit* action opens selected items in the <<../editor#,Content Editor>>. Each selected item opens in its own browser tab.

There are four ways to invoke Edit:

* Keyboard shortcut — `⌘ + e` on macOS, `ctrl + e` on Windows/Linux.
* The *Edit* button in the Content Navigator toolbar.
* Double-clicking an item in the Navigator grid.
* The *Edit* entry in the context menu (right-click an item).

For inherited content in a <<../layers#,layer>>, *Edit* is replaced with *Localize* — the same effect, but the action also severs data inheritance from the parent project for that item.
