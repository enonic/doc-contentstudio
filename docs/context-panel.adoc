= Context Panel
:toc: right
:imagesdir: context-panel/images

The *Context Panel* is the multi-function column on the right-hand side of the <<navigator#,Content Navigator>> and the <<editor#,Content Editor>>. It hosts a set of *widgets* — small tools that act on the currently selected content item. Widgets can be built-in, shipped by <<content-studio-plus#,Content Studio+>>, or contributed by any installed application.

Toggle the panel open and closed with the cog icon on the right of the toolbar.

image::widgets-panel-button.png[Context Panel toggle, 209]

The widget selector dropdown at the top of the panel lists the available widgets: the built-ins appear first, followed by widgets from installed applications. A filter field at the top narrows the list by keyword.

image::widgets-search.png[Widget selector with filter, 206]

== Built-in widgets

* <<context-panel/components#,Components>> — drag-and-drop component placement and configuration for renderable pages. Only available in the Content Editor for content with the <<editor/page-editor#,Page editor>> enabled.
* <<context-panel/details#,Details>> — properties and metadata of the selected content.
* <<context-panel/version-history#,Version history>> — all versions of the content, with compare and revert actions.
* <<context-panel/dependencies#,Dependencies>> — inbound and outbound references between content items.

== Content Studio+ widgets

The following widgets are contributed by <<content-studio-plus#,Content Studio+>>:

* <<layers#layers_panel,Layers>> — shows the status of a content item across every layer in the system.
* <<variants#,Variants>> — creates and manages A/B variants of the selected item.

== Custom widgets

The Context Panel is *pluggable*: any installed application can contribute a widget that appears in the widget selector. The interface for contributing a widget to the Context Panel is `contentstudio.contextpanel`; see the https://developer.enonic.com/docs/xp/stable/admin/widgets[XP admin widgets reference] for the descriptor format and lifecycle.
