= Content Studio+
:toc: right
:imagesdir: images

*Content Studio+* is a companion application that extends Content Studio with a set of advanced capabilities aimed at professional editorial teams. Content Studio itself is free and open source; Content Studio+ requires an Enonic subscription.

NOTE: This documentation covers both Content Studio and Content Studio+. Features that require Content Studio+ are marked with a *Content Studio+ feature* admonition at the top of the page or section where they are described.

== Installation

Content Studio+ is distributed as a standard Enonic application from https://market.enonic.com/vendors/enonic/content-studio-plus[Enonic Market]. Once the application is deployed to an Enonic XP instance with a valid subscription license, its features become available automatically inside Content Studio — no separate UI configuration is required.

== Features

The following capabilities in this documentation are provided by Content Studio+:

* <<ai#,AI Assistant (Juke)>> — in-context AI help for editors: field-level assistance, variants, mentions, and project-level instructions.
* <<archive#,Archive browser>> — the dedicated *Archive* main-menu entry for browsing, restoring, and permanently deleting archived content.
* <<variants#,Content Variants>> — create and manage variant copies of a content item for A/B testing.
* <<layers#ai_translation,AI translation>> — automatic translation of inherited content when localizing in a layer.
* <<layers#layers_panel,Layers widget>> — visualize the status of a content item across every layer in the system.

Individual features may also depend on additional applications (for example, the Juke AI translator app for AI translation). The page for each feature lists any extra requirements.

== Licensing

Content Studio+ requires a valid Enonic subscription. Without a license, the application can still be installed, but its UI contributions will not activate. See https://www.enonic.com/pricing[Enonic pricing] for subscription details.
